# Issue Tracker Webapp

This project was bootstrapped with Next.js 14+ and TypeScript.

## Getting Started

- Run `npm install` to install dependencies.
- Run `npm run dev` to start the development server.

## Structure

- `/app` - Main application routes and pages
- `/public` - Static assets
- `/components` - Reusable React components
- `/lib` - Utility libraries (Prisma, auth, CASL, etc.)
- `/prisma` - Prisma schema and migrations
- `/styles` - Global styles (Tailwind, etc.)
- `/api` - API route handlers (if not colocated in `/app/api`)

---

For more details, see the SRS and database schema in the backup folder.
